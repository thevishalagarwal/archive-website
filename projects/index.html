    Splice Site Prediction using Deep Learning (Ongoing)

       Prof. Ashish Anand, Dept. of CS, IIT Guwahati

    This project involves levaraging deep learning techniques for understanding the human genome. I am exploring various techniques for modelling of DNA sequences and discovering complex splicing signals and motifs. Splicing occurs at the junction of exon and intron and motifs are the part of DNA sequences which has some biological significance. Based on the task of splice site prediction, I am working on end-to-end sequence models with attentional mechanism for inferring important parts of DNA sequence which influences splicing and hence discover complex splicing signals. Also with an end-to-end approach, we want to learn motifs which influence splice site prediction.

    Deep Face Quality Assessment (Ongoing)

       Prof. Kannan Karthik, Dept. of EEE, IIT Guwahati

    Performance of facial recognition systems largely depends on quality of face image acquired for verification. The aim of the project is to develop an automatic Full Reference-based Face Image Quality predictor which evaluates a quality-check score for the image acquired and hence based on the score decide its utility for further verification.

    Filter Bank Generation using Incremental Spherical K-Means Clustering

       Prof. Prithwijit Guha, Dept. of EEE, IIT Guwahati

    The objective involved designing a novel online clustering pipeline inspired by the Spherical K-Means algorithm. One of the main motivation for this was big and very high-dimensional datasets for which knowing number of clusters apriori is not feasible. So we need a clustering algorithm which is 'dynamic'. In other words, the algorithm sees the data incrementally and then create, merge or split clusters based on data similarity. Further, we used the cluster's mean vector as a filter which represents the features of that cluster and hence, we get a bank of filters which can be used for various Computer Vision tasks. [Report]

    A Deep Learning Approach to Bone Age Estimation

       NVIDIA Niti Aayog Hackathon, Guwahati, India

    Implemented an end-to-end model for estimating the age of the bone from the X-Ray images of hand. The Inception-V3 architechture was used with a custom trainable regression layer at the output. We used transfer learning from the ImageNet model for better accuracy since the dataset was small. This project was done for the participation in Niti Aayog Hackathon. [Link]

    Multichannel Heart Sound Signal Acquisition and Segmentation

       Prof. Samarendra Dandapat, Dept. of EEE, IIT Guwahati

    Designed a hardware based two channel heart sound signal acquisition circuit which simultaneously reads heard sound signals from two auscultation areas. For demonstration, the signal obtained was displayed on an oscilloscope and also fed to a speaker system to aid 'lub-dub' heart sound hearing. Further, the signal obtained was fed to a computer where a GUI application was developed which gives additional information about the heart sound such as frequency domain analysis and heart sound rate in Beats Per Minute. This project was done as a part of Hardware Design Lab. [Report]
